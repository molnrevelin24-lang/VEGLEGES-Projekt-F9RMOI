Projekt címe: Reakcióidő játék

A program egy egyszerű reakcióidő-mérő játék Turtle grafikus ablakban.
A háttér véletlen idő után zöldre vált, ekkor kell reagálni.
Három próbálkozás után a program átlagos reakcióidőt számít és a játék újraindítható.
A játék végén az eredményt elmenti az eredmenyek.txt-be

Modulok és függvények:
-time (time.time())
-random (random.randint())
-turtle (write(), bgcolor(), goto(), onkey(), onscreenclick(), listen(), ontimer())

me_f9rmoi.py (ME_ido, ME_var, ME_mentes)
